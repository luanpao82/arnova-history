Download & Access

The ARNOVA Conference History Database is available in three formats. All files contain the same 5,406 presentation records spanning 2012–2025.

Data Files

JSON

Full structured data with nested author objects. Best for programmatic access and analysis.

Size: 6.4 MB

arnova_conference_catalog.json

Excel (.xlsx)

Flattened columns: Year, Edition, Location, Title, Authors, Affiliations, Presenter(s), Abstract, Track, Keywords.

Size: 1.7 MB

arnova_conference_catalog.xlsx

CSV

Comma-separated values with the same flattened column structure as Excel. Universal compatibility.

Size: 4.2 MB

arnova_conference_catalog.csv

Note: The downloadable data files include full author information (names, affiliations, emails). The build script and cached PDFs are in research/arnova_data/.

Data Schema

JSON Structure

The JSON file is an array of presentation objects. Each object has the following structure:

{
  "year": 2023,
  "conference_edition": "52nd",
  "location": "Orlando, FL",
  "title": "Presentation Title",
  "authors": [
    {
      "name": "Author Name",
      "affiliation": "University",
      "email": "email@university.edu",
      "presenter": true
    }
  ],
  "abstract": "Full abstract text...",
  "track": "Conference Track Name",
  "keywords": ["Qualitative", "Empirical"]
}

Rebuilding the Dataset

The dataset can be rebuilt from source PDFs using the build script:

# Install dependencies
pip install pdfplumber openpyxl

# Run the build script
python3 research/build_arnova_db.py

This will download all 14 catalog PDFs (if not already cached in arnova_data/), parse them, and regenerate the JSON, Excel, and CSV output files.
