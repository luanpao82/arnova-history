About

What is this?

This database compiles and structures all publicly available ARNOVA conference catalog data from 2012 to 2025 into an analyzable research dataset. Each entry includes the presentation title, abstract text, conference track, and methodology keywords (where available).

The data was extracted from 14 official ARNOVA conference catalog PDFs using automated parsing with manual quality review. The dataset enables longitudinal analysis of research trends and topical evolution in nonprofit and voluntary action scholarship.

Data Structure

Each record in the dataset contains the following fields:

FieldTypeDescriptionCoverage
yearintegerConference year100%
conference_editionstringEdition number (e.g., "52nd")100%
locationstringConference location100%
titlestringPresentation title100%
abstractstringPresentation abstract text82%
trackstringConference track classification21% (2021–2024)
keywordsarrayMethodology keywords21% (2021–2024)
